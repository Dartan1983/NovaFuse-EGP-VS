## Version History

### Versioning Convention
- **Major (X.0.0)**: Breaking changes to EGP protocol or verification semantics
- **Minor (X.Y.0)**: New features or non-breaking protocol extensions
- **Patch (X.Y.Z)**: Bug fixes, documentation updates, or legal clarifications

### Verification Semantics
All versions maintain **deterministic verification semantics** and **fail-closed security guarantees**. Patch releases do not alter verification outcomes, only improve documentation, legal precision, or tooling reliability.
